## 🐛 Известные проблемы

### Критичные
1. **Telegram bot handlers** - handler'ы возвращают заглушки вместо реальных данных
   - Файл: `telegram/bot.go`
   - Нужно: интегрировать с `_statsStore` и `_defaultProxy`
   
2. **Discord webhook** - не отправляет уведомления при событиях
   - Файл: `discord/webhook.go`, `main.go`
   - Нужно: вызвать `_discordWebhook` при событиях ARP

3. **Hotkeys** - не работают, не зарегистрированы
   - Файл: `hotkey/hotkey.go`, `main.go`
   - Нужно: создать глобальный экземпляр и обрабатывать события

### Средние
4. **WebSocket reconnect** - может терять соединение
   - Файл: `web/index.html`
   - Нужно: улучшить обработку ошибок

5. **ARP monitor** - не определяет hostname устройств
   - Файл: `stats/arp.go`
   - Нужно: добавить mDNS/NetBIOS запросы

### Мелкие
6. **Config validation** - нет проверки корректности конфигов
   - Файл: `cfg/config.go`
   - Нужно: добавить валидацию полей
