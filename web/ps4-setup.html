🎮 Настройки PS4

✅ Как подключить PS4:
  1. Подключите PS4 к ПК через Ethernet кабель
  2. На PS4: Настройки → Сеть → Настроить соединение с Интернетом
  3. Выберите "Легко" (автоматически)
  4. PS4 должна получить IP автоматически

Ручная настройка (если автоматическая не работает):

Параметр Значение
IP-адрес 192.168.100.100
Маска подсети 255.255.255.0
Шлюз 192.168.100.1
DNS 8.8.8.8
MTU 1472
