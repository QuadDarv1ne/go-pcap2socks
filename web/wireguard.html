⚙️ Конфигурация WireGuard

ℹ️ Информация: Заполните все поля для настройки WireGuard туннеля. Ключи должны быть в формате base64.
Ваш приватный ключ (генерируется через wg genkey)
Публичный ключ удаленного пира
Дополнительный ключ для аутентификации
Адрес удаленного сервера (host:port)
Ваш IP в туннеле
IP удаленной стороны в туннеле
IP диапазоны для маршрутизации через туннель

❓ Как использовать

1. Генерация ключей

Нажмите кнопку "Генерировать ключи" для создания приватного и публичного ключей.

2. Настройка сервера

Скопируйте публичный ключ и настройте удаленный сервер WireGuard.

3. Ввод конфигурации

Заполните все поля формы и нажмите "Сохранить конфигурацию".

4. Запуск туннеля

Нажмите "Запустить туннель" для активации подключения.

⚠️ Важно: Убедитесь, что удаленный сервер WireGuard настроен правильно и порт (по умолчанию 51820) открыт в фаерволе.
