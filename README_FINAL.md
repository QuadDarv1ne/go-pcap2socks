## 📋 Быстрый старт

### 1. Требования
- ✅ Windows 10/11
- ✅ Права администратора
- ✅ Npcap (https://npcap.com)
- ✅ Подключенный кабель Ethernet

### 2. Запуск

**Вариант A: Через скрипт (рекомендуется)**
```cmd
run.bat
```

**Вариант B: Вручную**
1. Правый клик на `go-pcap2socks.exe`
2. **Запуск от имени администратора**

### 3. Настройка PS4

**Автоматическая (DHCP):**
```
Настройки → Настройки сети → Настроить подключение к Интернету
Кабель (LAN) → Автоматически
```

**Ручная (если автоматическая не работает):**
```
IP-адрес:     192.168.137.100
Маска:        255.255.255.0
Шлюз:         192.168.137.1
DNS 1:        8.8.8.8
DNS 2:        1.1.1.1
MTU:          1486
Прокси:       Не использовать
```

## ❓ Решение проблем

### Сервис не запускается
1. Запустите **от имени администратора**
2. Установите **Npcap**: https://npcap.com
3. Проверьте **кабель Ethernet**
4. Проверьте логи: `app.log`

### Ошибка "network adapter disconnected"
1. Проверьте кабель Ethernet
2. Проверьте, что адаптер включён
3. Перезапустите сервис

### NAT тип 3
1. Включите UPnP на роутере
2. Пробросьте порты: **3478-3480 TCP/UDP** → `192.168.137.100`
