# go-pcap2socks Документация

## Обзор

**go-pcap2socks** — это мощный прокси-инструмент, который перенаправляет сетевой трафик с любого устройства на SOCKS5 прокси. Он функционирует как маршрутизатор, позволяя подключать различные устройства к SOCKS5 прокси-серверу.

### Поддерживаемые устройства

- **Игровые консоли**: Xbox, PlayStation (PS4, PS5), Nintendo Switch
- **Мобильные устройства**: iOS, Android телефоны и планшеты
- **Компьютеры**: Windows, macOS, Linux
- **Другие устройства**: Принтеры, Smart TV, IoT устройства

### Ключевые возможности

- 🎮 Маршрутизация трафика с любого устройства через SOCKS5 прокси
- 🔧 Гибкие правила маршрутизации на основе IP, порта и протокола
- 🌐 Поддержка DNS с несколькими серверами
- 📝 Подробное логирование и возможности отладки
- 🚀 Высокопроизводительный стек TCP/IP (на базе gVisor)
- ⚙️ Простая конфигурация через JSON файл

## Быстрый старт

```bash
# Установка
go install github.com/DaniilSokolyuk/go-pcap2socks@latest

# Настройка (открывает конфиг в редакторе)
go-pcap2socks config

# Запуск (требуются права администратора)
sudo go-pcap2socks
```

  "pcap": {

  },
  "dns": {

## Настройка сети

При запуске go-pcap2socks отображает настройки сети, которые нужно применить на вашем устройстве:

```
Настройте ваше устройство со следующими параметрами сети:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  IP адрес:       172.26.0.2 - 172.26.255.254
  Маска подсети:  255.255.0.0
  Шлюз:           172.26.0.1
  MTU:            1486 (или меньше)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Шаги настройки устройства

1. **Откройте настройки сети** на вашем устройстве
2. **Установите статический IP** из указанного диапазона (исключая IP шлюза)
3. **Установите маску подсети** как указано
4. **Установите шлюз** равным localIP из конфигурации
5. **Установите MTU** если требуется (особенно для Nintendo Switch и PS5)

## Устранение неполадок

### Проблемы с правами доступа

- **Windows**: Запустите от имени администратора (правый клик → Запуск от имени администратора)
- **macOS/Linux**: Используйте `sudo go-pcap2socks`

### Распространённые проблемы

1. **Ошибка "Permission denied"**
   - Решение: Запустите с правами администратора/root

2. **Устройство не может подключиться**
   - Проверьте настройки брандмауэра
   - Убедитесь в правильности настройки IP на устройстве
   - Убедитесь, что go-pcap2socks запущен

3. **Медленное соединение**
   - Проверьте скорость вашего SOCKS5 прокси
   - Уменьшите значение MTU
   - Проверьте загруженность сети

### Настройка игровых консолей

Для **Nintendo Switch** и **PS5**:
- Необходимо вручную установить значение MTU в настройках сети
- Требуемое значение MTU отображается при запуске go-pcap2socks
- Используйте точно указанное значение или меньше
