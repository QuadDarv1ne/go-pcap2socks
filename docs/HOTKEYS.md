# Горячие клавиши в go-pcap2socks

## Обзор

go-pcap2socks поддерживает глобальные горячие клавиши для быстрого управления сервисом без открытия веб-интерфейса.

## 🔧 Настройка

### Включение горячих клавиш

В `config.json`:

```json
{
  "hotkey": {
    "enabled": true,
    "toggle": "Ctrl+Alt+P"
  }
}
```

### Параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `enabled` | bool | `true` | Включить горячие клавиши |
| `toggle` | string | `Ctrl+Alt+P` | Комбинация для переключения прокси |

## ⌨️ Доступные комбинации

### По умолчанию

| Комбинация | Действие | Описание |
|------------|----------|----------|
| `Ctrl+Alt+P` | Toggle Proxy | Переключение режима прокси |
| `Ctrl+Alt+R` | Restart | Перезапуск сервиса |
| `Ctrl+Alt+S` | Stop | Остановка сервиса |
| `Ctrl+Alt+L` | Toggle Logs | Переключение видимости логов |

### Настройка комбинаций

Для изменения горячих клавиш отредактируйте `config.json`:

```json
{
  "hotkey": {
    "enabled": true,
    "toggle": "Ctrl+Shift+P",
    "restart": "Ctrl+Shift+R",
    "stop": "Ctrl+Shift+S",
    "logs": "Ctrl+Shift+L"
  }
}
```

## 📋 Действия

### Toggle Proxy (`Ctrl+Alt+P`)

Переключает режим прокси:
- SOCKS5 → Direct
- Direct → SOCKS5

**Уведомление:**
```
Горячие клавиши
Переключение прокси
```

### Restart (`Ctrl+Alt+R`)

Перезапускает сервис go-pcap2socks.

**Уведомление:**
```
Горячие клавиши
Перезапуск сервиса
```

⚠️ **Внимание:** Перезапуск занимает 5-10 секунд. В это время трафик не обрабатывается.

### Stop (`Ctrl+Alt+S`)

Останавливает сервис.

**Уведомление:**
```
Горячие клавиши
Остановка сервиса
```

⚠️ **Внимание:** После остановки трафик не обрабатывается до следующего запуска.

### Toggle Logs (`Ctrl+Alt+L`)

Показывает/скрывает окно логов (если доступно).

**Уведомление:**
```
Горячие клавиши
Переключение логов
```

## 🔔 Уведомления

При использовании горячих клавиш показываются системные уведомления:

### Типы уведомлений

| Тип | Значок | Когда |
|-----|--------|-------|
| Info | ℹ️ | Toggle Proxy, Toggle Logs |
| Warning | ⚠️ | Stop |
| Error | ❌ | Ошибка выполнения |

### Отключение уведомлений

В Windows:
1. Параметры → Система → Уведомления
2. Найдите "go-pcap2socks"
3. Отключите уведомления

Или в `config.json`:

```json
{
  "notify": {
    "enabled": false
  }
}
```

## 🎯 Сценарии использования

### Сценарий 1: Быстрое отключение прокси

**Проблема:** Нужно быстро отключить прокси для доступа к локальным ресурсам.

**Решение:**
1. Нажмите `Ctrl+Alt+P`
2. Прокси переключится в режим Direct
3. Локальные ресурсы доступны напрямую

### Сценарий 2: Перезапуск при проблемах

**Проблема:** Прокси перестал обрабатывать трафик.

**Решение:**
1. Нажмите `Ctrl+Alt+R`
2. Сервис перезапустится
3. Подождите 5-10 секунд
4. Проверьте работу

### Сценарий 3: Экстренная остановка

**Проблема:** Нужно немедленно остановить обработку трафика.

**Решение:**
1. Нажмите `Ctrl+Alt+S`
2. Сервис остановится
3. Трафик пойдёт напрямую

## 💻 PowerShell управление

### Проверка статуса горячих клавиш

```powershell
# Проверить config.json
Get-Content config.json | ConvertFrom-Json | Select-Object -ExpandProperty hotkey
```

### Включение/отключение

```powershell
# Включить горячие клавиши
$config = Get-Content config.json | ConvertFrom-Json
$config.hotkey.enabled = $true
$config | ConvertTo-Json -Depth 10 | Out-File config.json -Encoding UTF8

# Отключить горячие клавиши
$config.hotkey.enabled = $false
```

### Изменение комбинации

```powershell
$config = Get-Content config.json | ConvertFrom-Json
$config.hotkey.toggle = "Ctrl+Shift+P"
$config | ConvertTo-Json -Depth 10 | Out-File config.json -Encoding UTF8
```

## 🐛 Troubleshooting

### Горячие клавиши не работают

**Причины:**
1. Горячие клавиши отключены в config.json
2. Приложение запущено без прав администратора
3. Комбинация занята другим приложением

**Решение:**
```powershell
# Проверить настройки
Get-Content config.json | ConvertFrom-Json | Select-Object -ExpandProperty hotkey

# Перезапустить от администратора
Start-Process powershell -Verb RunAs -ArgumentList "-Command cd M:\GitHub\go-pcap2socks; .\go-pcap2socks.exe"
```

### Конфликт комбинаций

Если комбинация занята другим приложением:

1. Откройте config.json
2. Измените `toggle` на другую комбинацию
3. Сохраните файл
4. Перезапустите сервис

```json
{
  "hotkey": {
    "toggle": "Ctrl+Alt+F12"
  }
}
```

### Уведомления не показываются

**Причины:**
1. Уведомления отключены в Windows
2. Отключены в config.json

**Решение:**
```powershell
# Проверить настройки уведомлений
Get-Content config.json | ConvertFrom-Json | Select-Object -ExpandProperty notify

# Включить уведомления
$config = Get-Content config.json | ConvertFrom-Json
$config.notify.enabled = $true
$config | ConvertTo-Json -Depth 10 | Out-File config.json -Encoding UTF8
```

## 🔒 Безопасность

### Ограничение доступа

Горячие клавиши работают только когда:
- Приложение запущено от имени администратора
- Веб-интерфейс активен
- Сервис работает

### Защита от случайного нажатия

Используйте комбинации с 3 клавишами:
- `Ctrl+Alt+P` вместо `Ctrl+P`
- `Ctrl+Shift+Alt+S` вместо `Ctrl+S`

